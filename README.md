# StudyMate - Academic Support Platform

StudyMate is a comprehensive platform designed to assist students in managing their academic life, find career opportunities, and collaborate on study materials.

## Project Overview

The platform is built to solve common university issues, providing:
- **Personalized Dashboards**: Customized student experience.
- **Career Guidance**: Bridge between students and internships.
- **Academic Issue Resolution**: Seamless tracking and solving of academic problems.
- **Study Materials**: Centralized repository for all academic resources.

## Tech Stack
- **Frontend**: React.js
- **Backend**: Node.js & Express
- **Database**: MongoDB
