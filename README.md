# StudyMate - Smart Study and Academic Issue Resolution System

## Project Overview
StudyMate is a comprehensive academic platform designed to streamline the educational experience for students, lecturers, and administrators. It integrates personalized dashboard management, study materials handling, internship and career guidance, and a robust academic issue resolution system into a unified web application.

## Technology Stack
* **Frontend:** React.js
* **Backend:** Node.js, Express.js
* **Database:** MongoDB
* **Styling:** CSS3

## Run Locally
1. Clone the repository
2. Run `npm install` inside both `frontend/` and `backend/` directories
3. Set up the `.env` variables for database connection
4. Run the frontend and backend servers.
